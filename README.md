# Setup

## install
## 開発環境に必要なコマンド

VSCode Extentionsの検索欄で `@recommended` で検索して出てきたものを入れる

- cpplint
    - `pip install cpplint`
- clang-tidy
    - `sudo apt install clang-tidy`
- clang-format
    - `sudo apt install clang-format`

## 最初に呼ぶコマンド

```bash
make setup
```


## VSCode

`Ctrl + Shift + X`(Search Extentions)  ->  `@recommended` -> install all
